Guía práctica: Creación y animación de personajes 3D con IA

En esta guía te enseñaremos cómo crear, modelar y animar un personaje 3D desde cero usando herramientas impulsadas por inteligencia artificial. ¡No necesitas experiencia previa en diseño 3D!

1. Crear el prompt

Para comenzar, necesitas una descripción clara de tu personaje. Por ejemplo:

Prompt: "un detective, brazos extendidos, t pose, estilo 3d, caricatura, posicion firme neutra de pie, fondo blanco, kawaii".

Este texto se usa para generar la imagen del personaje.

2. Generar la imagen del personaje

Usa el generador de imágenes de Bing para crear la imagen inicial de tu personaje.

Ir al generador de imágenes

3. Generar el modelo 3D

Convierte la imagen generada en un modelo 3D utilizando Stable Fast 3D.

Generar modelo 3D

Esto creará un archivo en formato GLB que puedes usar para animar.

4. Convertir el modelo 3D de GLB a FBX

Si necesitas convertir el modelo a un formato compatible con Mixamo, utiliza un convertidor en línea:

Convertir a FBX

5. Animar el personaje

Con el archivo en formato FBX, sube tu modelo a Mixamo para agregar animaciones prediseñadas.

Animar en Mixamo

Resultado Final

Al combinar todos los pasos, obtendrás un personaje animado que podrás usar en tus proyectos.

¡Sigue experimentando y aprendiendo!
